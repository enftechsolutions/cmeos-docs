# Checklist de Preparo

A tela de preparo guia você por todas as etapas de inspeção e embalagem. O sistema só permite finalizar quando todas as etapas obrigatórias forem concluídas.

---

## Passo a passo do preparo

**1. Inspeção visual**

Verifique se o material está:
- Livre de **sujidade visível**, resíduos orgânicos ou manchas
- Livre de **oxidação** (ferrugem, manchas escuras em metais)
- Com a **superfície íntegra** (sem lascas, rachaduras ou dobras indevidas)

Marque as caixas de seleção para confirmar cada item da inspeção. Você pode clicar em **Marcar Todos** para agilizar quando o material estiver em boas condições.

**2. Teste funcional**

Para instrumentais articulados (tesouras, pinças, porta-agulhas), verifique:
- Articulações funcionando com facilidade
- Corte eficiente (tesouras)
- Travas funcionando corretamente

Marque cada item testado.

**3. Conferência de composição (somente para caixas e kits)**

Esta etapa aparece apenas para **caixas cirúrgicas** ou **kits com composição definida**.

Para caixas com lista de peças fixas, a tela mostra cada instrumento que deve estar presente. Você pode bipar o QR code de cada peça ou marcar manualmente a caixa de seleção. Uma barra de progresso mostra o percentual de conferência concluído.

Para kits por composição, a tela mostra o instrumento e a quantidade mínima necessária. Bipe ou confirme até atingir a quantidade. O contador atualiza em tempo real.

> **Atenção:** Se algum instrumento estiver faltando, você deve registrar nas **Observações** (etapa 6) antes de poder finalizar. O sistema exige justificativa para composições incompletas.

**4. Foto com integrador químico**

Tire uma foto da embalagem montada com o **integrador químico** visível. O integrador é o indicador que comprova que o processo de esterilização foi realizado corretamente.

Clique na área de upload, tire a foto ou selecione o arquivo da galeria.

**5. Selecione o tipo de embalagem**

Escolha o material de embalagem utilizado. Cada opção tem uma validade pós-esterilização diferente, calculada automaticamente pelo sistema:

| Embalagem | Validade |
|-----------|---------|
| Papel grau cirúrgico (selado) | 180 dias |
| Campo duplo (tecido) | 30 dias |
| Caixa metálica com filtro | 6 meses |
| Embalagem rígida | Conforme fabricante |

A validade exata será impressa na etiqueta.

**6. Observações (opcional / obrigatório se houver pendência)**

Use este campo para registrar instrumentos danificados ou faltando, condições especiais do material, ou qualquer informação relevante para o próximo operador.

Se a composição estiver incompleta, este campo torna-se **obrigatório**.

**7. Finalizar ou Rejeitar**

- **Finalizar Preparo:** O material é aprovado, etiqueta gerada e item enviado para a fila de esterilização.
- **Rejeitar Material:** O item é devolvido ao Expurgo para nova lavagem. O motivo da rejeição deve ser informado.

---

## O que acontece após finalizar?

1. O sistema gera automaticamente os dados da **etiqueta de rastreabilidade**
2. Uma tela de sucesso é exibida com a **prévia da etiqueta**
3. Você pode **imprimir** a etiqueta diretamente pelo modal de impressão
4. O material avança na fila para a **Esterilização**
