# Usuários e Permissões

<Callout type="warning">
  Esta área é exclusiva para **Administradores**. Apenas o administrador pode criar, editar ou desativar usuários.
</Callout>

---

## Como criar um novo usuário

<Steps>

### Acesse Configurações → Usuários

### Clique em "Novo Usuário"

### Preencha os dados

- **Nome completo**
- **Login** (nome de usuário para acesso)
- **E-mail** (para recuperação de senha)
- **Perfil de acesso** (veja tabela abaixo)
- **Setor** ao qual pertence
- **Registro profissional** (COREN, CRM — se aplicável)
- **Senha inicial** (o usuário poderá alterar no primeiro acesso)

### Salve

O usuário receberá acesso imediato ao sistema.

</Steps>

---

## Perfis de acesso disponíveis

| Perfil | Acesso |
|--------|--------|
| **Administrador** | Acesso total: configurações, usuários, relatórios |
| **Responsável CME** | Todos os módulos operacionais + indicadores |
| **Operador CME** | Expurgo, Preparo, Esterilização, Desinfecção, Estoque |
| **Setor Consumidor** | Solicitar materiais e consultar status |
| **Visualizador** | Somente leitura (auditoria e estoque) |

---

## Desativar um usuário

Para colaboradores que saíram da instituição:

1. Localize o usuário na lista
2. Clique em **Editar**
3. Altere o status para **Inativo**
4. Salve

O usuário perde o acesso imediatamente, mas seu histórico de ações é preservado para fins de rastreabilidade.

<Callout type="info">
  Nunca delete um usuário — apenas desative. Isso garante que o histórico de ações registradas por ele permaneça vinculado ao seu nome.
</Callout>
